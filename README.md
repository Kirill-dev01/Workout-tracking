# Домашнее задание: Учёт тренировок

React-приложение для учёта пройденных километров с использованием TypeScript.

## Реализованный функционал:
* Добавление новых тренировок с автоматической сортировкой по дате (по убыванию).
* Суммирование дистанции, если запись на выбранную дату уже существует.
* Удаление записей из списка.
* Редактирование существующих записей (перенос данных обратно в форму с удалением старой строки).
* Строгая типизация компонентов и событий.
